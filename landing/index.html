FAQ

Good questions.

Which podcasts can I track?

Any YouTube channel that publishes long-form episodes with captions. Add channels by pasting their link in Telegram or the config file.

Do I need a YouTube API key?

No. Mindmonk polls channels directly — no Google API key, no setup beyond Telegram.

How often does it check for new episodes?

Every 30 minutes by default. Only brand-new uploads trigger a brief — once processed, it won’t repeat.

What if an episode has no transcript?

It’s skipped (optionally with a heads-up). Mindmonk works with captioned episodes, which covers the vast majority of long-form podcasts.
